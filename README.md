# Description

Test Api NestJs.

[Swagger](http://localhost:3000/api)

## Installation

```bash
# install 
make i

# postgre
make docker-compose
```

## Running the app

```bash
# development
$ make start

# watch mode
$ make dev

# production mode
$ make prod
```

## Test

```bash
# unit tests
$ make test_unit

# e2e tests
$ make test_e2e
```
